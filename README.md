# Chat application

Realtime chat app with websockets using Node.js, Express and Socket.io with Vanilla JS on the frontend with a custom UI.

## ***Usage***

- npm install

- npm run dev

- Go to localhost:3001

## ***Description:-***

 ***server.js:***

- regiesterd event

    1-join room

    2-chatMessage

- fired  event:

    1-roomUsers

    2-message

 ***main.js***

- regiesterd event:

    1-roomUsers

    2-message

- fired  event:

    1-joinRoom

    2-chatMessage
